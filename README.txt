📖 简介
───────
本软件基于中国传统命理学九大经典典籍，结合 AI 大模型
进行专业的四柱八字命理分析。

参考典籍：《穷通宝典》《三命通会》《滴天髓》《渊海子平》
《千里命稿》《协纪辨方书》《果老星宗》《子平真诠》《神峰通考》

🚀 使用方法
───────────
1. 双击「启动.bat」或「赛博算命.exe」
2. 看到启动提示后，打开浏览器访问 http://localhost:3456
3. 首次使用先在页面左侧「⚙ API 设置」中填入你的 API Key
4. 填写出生信息，点击「提交基本信息」开始排盘
5. 右侧对话区与 AI 深度探讨命理

🔑 获取 API Key
───────────────
本软件需要 AI 大模型 API Key 才能进行分析。推荐：

  DeepSeek (推荐，便宜好用):
  https://platform.deepseek.com  → 注册 → API Keys → 创建

  也支持 OpenAI / 通义千问 / 硅基流动 等兼容接口。
  只需修改 API 地址和模型名称即可。

⚙️ 配置说明
───────────
方式1：在网页左侧「⚙ API 设置」面板直接填写（推荐）
        API Key 仅保存在你的浏览器本地

方式2：修改同目录下的 config.json 文件：
  {
    "port": 3456,
    "apiKey": "sk-你的API Key",
    "model": "deepseek-v4-pro",
    "baseUrl": "https://api.deepseek.com"
  }

方式3：设置环境变量：
  BAZI_API_KEY=sk-你的Key
  BAZI_MODEL=deepseek-v4-pro
  BAZI_BASE_URL=https://api.deepseek.com

⚠️ 注意事项
───────────
• 首次启动可能被 Windows 防火墙拦截，请允许访问
• 如端口 3456 被占用，修改 config.json 中的 port 值
• 按 Ctrl+C 可停止服务器
• 命理分析仅供文化研究参考，人生在于自身的努力和选择

🔧 技术支持
───────────
如遇问题，检查：
1. 浏览器能否打开 http://localhost:3456
2. API Key 是否正确且有余额
3. 防火墙是否拦截了 3456 端口
4. 控制台窗口是否有错误提示
